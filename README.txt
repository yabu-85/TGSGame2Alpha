現在制作中の2人対戦のシューティングゲームです

PC		 / コントローラー
コントローラー 	 / コントローラー
の組み合わせで対戦が可能で
OPTION（Tabキー）から設定が可能です

制作人数：１人
制作期間：4か月
開発言語：C / C++ / HLSL
ライブラリ：DirectX11 / FBXSDK

-操作方法　PC-
WASD　　　　　：移動
左クリック    ：射撃
右クリック　　：覗き込み
Space　　　　 ：ジャンプ
TABキー　　   ：Pauseメニュー

-操作方法　パッド-
左スティック　：移動
右トリガー　　：射撃
左トリガー　　：覗き込み
Bボタン　　　 ：ジャンプ
Startボタン　 :Pauseメニュー
Escキーで強制終了できます
